# ecommerce store

1. Cloner le projet
2. ``` npm install ```
3. ``` npm run build ```
4. Ouvrir dist/pages, dist/components ou dist/elements

# Front-end specs :

| Device     | Breakpoint    |
| --------|---------|
| Mobile  | < 768px   |
| Tablette portrait, paysage, et petit desktop | >= 768px <= 1024px |
| Desktop | Full frame |
